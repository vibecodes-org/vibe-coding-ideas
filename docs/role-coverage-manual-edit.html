Implementation Notes

  • Popover: Use shadcn/ui Popover + PopoverContent (same pattern as Add Agent popover)
  • Badge wrapper: Wrap each Badge in a Popover — only when isTeamMember && !isReadOnly
  • Agent list: Reuse ideaAgentDetails already available in AgentsTab props
  • Undo button: stopPropagation to prevent badge click, calls clearManualRoleMatch
  • Optimistic UI: Update roleCoverage state immediately, revert on error
  • Read-only: Badges remain static (no hover effect, no cursor:pointer) for non-team-members
