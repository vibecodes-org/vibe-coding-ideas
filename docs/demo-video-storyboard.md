# VibeCodes Demo Video Storyboard

**Duration**: 90 seconds
**Format**: Screen recording, no voiceover, on-screen annotations
**Resolution**: 1920x1080 (record at 2x and scale to 1080p for clarity)
**Theme**: Dark mode
**Browser**: Chrome (clean profile, no extensions visible, no bookmarks bar)

---

## Pre-recording Setup

- [ ] Log into VibeCodes with a demo account that has existing ideas and agents
- [ ] Ensure dark theme is active
- [ ] Clear notification badges
- [ ] Prepare an idea title and brief description to type during recording
- [ ] Ensure the dev server is running (`npm run dev`)
- [ ] Hide browser bookmarks bar and extension icons
- [ ] Set browser zoom to 100%
- [ ] Close all other tabs

---

## Storyboard

### Scene 1: Landing Page (0:00 - 0:10)

**Shot**: Landing page at `vibecodes.co.uk`
**Action**:
1. Brief pause on hero section (2s)
2. Slow scroll to show features grid (3s)
3. Click "Get Started" CTA button

**Annotation**: _"VibeCodes — AI-Powered Idea Board"_
**Transition**: Page navigates to auth/dashboard

---

### Scene 2: Create a New Idea (0:10 - 0:25)

**Shot**: Dashboard, then New Idea form
**Action**:
1. Click "New Idea" button in nav bar
2. Type idea title: `"Build a CLI Dashboard for DevOps Monitoring"`
3. Type 1-2 sentence description
4. Click "Enhance with AI" button
5. Wait for AI streaming to complete (show the enhancement happening)
6. Accept the enhanced description
7. Submit the idea

**Annotation**: _"AI enhances your idea description"_ (during streaming)
**Key moment**: The streaming AI enhancement is the hero shot — let it play in real-time for 3-4 seconds

---

### Scene 3: AI Board Generation (0:25 - 0:40)

**Shot**: Idea detail page
**Action**:
1. Brief pause on the idea detail page showing description, tags, team (2s)
2. Click "Board" button to navigate to the board
3. Click "AI Generate" button to generate tasks
4. Wait for AI to stream generated tasks into columns

**Annotation**: _"AI generates a full task board from your idea"_
**Key moment**: Tasks appearing in columns — show 3-4 seconds of the generation

---

### Scene 4: Kanban Board Interaction (0:40 - 0:55)

**Shot**: Populated kanban board
**Action**:
1. Pan across the board showing populated columns (2s)
2. Drag a task card from "To Do" to "In Progress" (smooth, deliberate drag)
3. Click on a task card to open the detail dialog
4. Show: labels, checklist items, due date picker, assignee selector
5. Close the dialog

**Annotation**: _"Full project management — drag-and-drop, labels, checklists"_

---

### Scene 5: AI Agent Integration (0:55 - 1:15)

**Shot**: Agents Hub, then board
**Action**:
1. Navigate to `/agents` (Agents Hub)
2. Brief pan across agent cards showing roles and skills (3s)
3. Navigate back to the board
4. Click a task's assignee dropdown
5. Select an AI agent from the dropdown
6. Show the agent assigned with their role badge

**Annotation**: _"AI agents join your team as first-class members"_

**Then**:
1. Quick cut to the MCP Integration guide page (`/guide/mcp-integration`)
2. Show the `claude mcp add` command
3. Brief pause (2s)

**Annotation**: _"54 MCP tools — connect in one command"_

---

### Scene 6: Wrap-up & End Card (1:15 - 1:30)

**Shot**: Board view with progress
**Action**:
1. Return to the board showing tasks distributed across columns
2. Slow zoom out showing the full board state
3. Fade to end card

**End card** (final 5 seconds):
- VibeCodes sparkle icon (centered)
- "VibeCodes" in large text
- "vibecodes.co.uk" below
- "Get Started Free" CTA text

## Annotation Style

| Property | Value |
|----------|-------|
| Font | System sans-serif (Inter or Geist if available) |
| Color | White (#ffffff) |
| Background | Semi-transparent dark (#09090b at 80% opacity) |
| Position | Bottom-center, 60px from bottom edge |
| Padding | 12px 24px |
| Border radius | 8px |
| Duration | 3-4 seconds per annotation |
| Animation | Fade in (200ms) / fade out (200ms) |

## Post-production Notes

- **Speed up typing**: All typing sequences at 2x speed
- **Cursor highlight**: Add a subtle spotlight/highlight around the cursor
- **Transitions**: Use simple crossfade (300ms) between scenes
- **Background music**: Upbeat, lo-fi instrumental (royalty-free). Suggestions:
  - Artlist: search "tech" or "startup" category
  - YouTube Audio Library: "Chill" or "Inspiring" mood
- **Export settings**: MP4, H.264, 1080p, target < 50MB
- **Thumbnail**: Use the kanban board screenshot with "VibeCodes Demo" text overlay
