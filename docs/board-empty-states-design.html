Implementation Notes

Tailwind classes used: All designs use existing Tailwind utilities — no custom CSS needed.

  • Gradient glow: absolute inset-0 rounded-full bg-gradient-to-br from-{color}-500/30 to-{color}-500/30 blur-xl
  • Gradient heading: bg-gradient-to-r from-{color}-300 to-{color}-300 bg-clip-text text-transparent
  • Gradient button: bg-gradient-to-r from-{color}-600 to-{color}-600 hover:from-{color}-500 hover:to-{color}-500 text-white shadow-lg shadow-{color}-500/20
  • Nudge cards: rounded-lg border border-{color}-500/20 bg-{color}-500/5 hover:border-{color}-500/40 hover:bg-{color}-500/10
  • Tinted info box: rounded-lg border border-amber-500/15 bg-amber-500/5
