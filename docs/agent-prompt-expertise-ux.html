3. Option A — Interaction Detail

Default state (collapsed)

7 role chips + 1 "More…" chip in the existing 4×2 grid. Clicking "More…" reveals the expanded row below with a dashed separator.

Expanded state

Additional 7 chips appear below the separator in the same 4-column grid (2 rows, last row has 3 chips). The "More…" chip changes to "Less ↑" to collapse. State persists in sessionStorage so returning users see their preference.

Selection

Clicking any chip (visible or expanded) works identically — populates role, goal, expertise, constraints, approach. Selected chip gets the violet highlight. Only one chip can be selected at a time.

Mobile (<640px)

Grid falls back to 3 columns. Default: 3×2 + "More…" (7 chips total). Expanded: additional 3×3 rows.

Accessibility

  • "More…" chip: aria-expanded="false", toggles to "true"
  • Expanded section: role="group" with aria-label="Additional role templates"
  • Focus management: expanding moves focus to first new chip
