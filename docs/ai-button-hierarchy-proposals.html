Compass Recommendation

Option B (Smart CTA) is the strongest choice. Lowest effort, solves hierarchy + double-click, clean post-board state. Power users have the header "Enhance" button; new users get guided naturally by the morphing CTA.

Option A is best for onboarding-heavy products but risks annoying returning users. The locked steps could feel patronising.

Option C is most visually appealing with maximum freedom. Best if Enhance and Generate are truly independent features.

Hybrid Possibility: B + C Elements

Take Option B's contextual smart CTA, but borrow Option C's tip text instead of the full stepper. Lighter informational footprint, same guided progression.
