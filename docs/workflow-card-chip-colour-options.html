Workflow “idle” chip — colour / style options

The idle chip (filled Play + workflow name + done/total + thin progress bar) reads as a loud CTA today because of the solid bg-blue-600 fill with white text. Below: the current treatment plus six calmer alternatives, each rendered as a live mini board-card in context, at two fill levels, with a long-name truncation example on a couple. Pick whichever feels right — this is your call.

Recommendation: Option 1 — Tinted / translucent (border border-blue-500/25 bg-blue-500/15 text-blue-400). It is the same idiom already used by the active / approval / failed states on this card, so the idle state becomes a quiet member of one consistent family instead of a standalone CTA. Pair it with the softened bar (track bg-blue-500/15 + fill bg-blue-500/70). If you want it even quieter, Option 6; if you want the pill itself to recede entirely and let the bar carry the colour, Option 5. All options below meet WCAG AA except where flagged.
