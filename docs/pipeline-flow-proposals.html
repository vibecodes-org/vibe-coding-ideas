Implementation Notes

Reusable Component

Both options would be built as a single <PipelineFlow /> React component with a variant prop ("steps" | "animated"). Accepts optional activeStep for contextual highlighting on guide pages.

Placement

Landing page: replaces the current 4-step "How it works" section. Guide index: above the guide cards grid. First-run dashboard: replaces or augments the 5-step progress bar.

Responsive

Both options stack vertically on mobile. Option A rotates the connector line. Option B rotates the arrows. Cards compress to narrower widths with smaller icons.

Recommendation

Option B (Animated Pipeline) is the stronger choice. The sequential reveal creates a narrative ("this leads to this leads to this") that directly addresses the mental model gap identified in the UX audit. Option A is better for static/documentation contexts.
