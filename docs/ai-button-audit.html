Tier Definitions Visual Reference

Tier 1
Primary AI Action
Buttons that generate or apply AI content as the main action. These are the most prominent AI buttons on a page.
Used for: Generate Board Generate Tasks Create & Generate Board Apply N Tasks
Generate Board Apply 5 Tasks
Style: bg-violet-600 text-white hover:bg-violet-700 · Icon: Sparkles h-4 w-4 · Size: default or lg
Tier 2
Dialog Action
AI action buttons inside dialogs/modals that advance through an AI-powered flow (e.g., enhance steps).
Used for: Enhance Enhance with Answers Apply Enhancement Next
Enhance Apply Enhancement
Style: bg-violet-600 text-white hover:bg-violet-700 · Icon: Sparkles h-4 w-4 · Size: default
Tier 3
Inline Trigger
Buttons in toolbars that open an AI dialog or start an enhancement flow. Visually subordinate to the content they serve.
Used for: Enhance with AI toolbar buttons · AI Generate toolbar button
Enhance with AI 3 AI Generate 5
Style: variant="outline" size="sm" · Icon: Sparkles h-3.5 w-3.5 · Credit badge: bg-violet-600 rounded-full
Tier 4
Ghost / Inline
Subtle enhance buttons embedded within editors or content areas. Minimal visual footprint to avoid distracting from content.
Used for: Task description enhance · Discussion enhance
Enhance
Style: variant="ghost" size="sm" · Icon: Sparkles h-3 w-3 · Label: always "Enhance" (short)
Special Cases (Keep As-Is)
  • Onboarding CTA — The card-style "Generate your first board with AI" CTA on the first-run dashboard has a unique visual treatment (large card with icon and description). This is intentional for the onboarding context and should not be standardized.
  • Idea form toolbar — The gradient-border "Enhance with AI" button in the create/edit idea form uses a custom animated gradient border style. This unique toolbar context warrants keeping its distinctive appearance.

Design principles behind the tier system:
  • Visual hierarchy -- Tier 1/2 (violet solid) commands attention for primary actions. Tier 3 (outline) is present but subordinate. Tier 4 (ghost) is discoverable without competing for attention.
  • Consistent AI signal -- Every AI button uses the Sparkles icon (only size varies by tier). Violet is the universal "AI" color throughout the app.
  • Credit badge uniformity -- All credit indicators use the same bg-violet-600 text-white rounded-full px-1.5 text-[10px] pill format regardless of context.
  • No over-standardization -- The two special cases (onboarding CTA and gradient toolbar button) are intentionally excluded because their unique visual treatments serve their specific contexts well.
