Workflow Alternative Mockups v2

Approach A — Idea-Level Templates + Role-Based Agent Assignment

Core idea: Templates live on the idea, not on agent profiles. Template steps declare roles (e.g. "Business Analyst", "Developer"). When a template is applied to a task, roles are auto-matched against the idea's agent pool. No orchestrator/worker type system needed.

Removed

  • - Worker / Orchestrator type toggle
  • - Conditional fields per agent type
  • - Templates stored on agent profiles
  • - "Orchestration Agent" picker on conversion

Added

  • + Idea-scoped workflow templates
  • + Role-annotated template steps
  • + Auto-matching roles to agent pool
  • + Template application preview dialog
